# fvtt-ptbr-sf2e-translation

Módulo de tradução em português do Brasil para **Starfinder Second Edition** no **Foundry VTT**.

## Status

Estrutura inicial preparada para publicação do projeto e integração com o Crowdin.

## Observações importantes

Antes de publicar o módulo para uso real, ajuste estes pontos:

1. O campo `REPLACE_WITH_REAL_SYSTEM_ID` no `module.json` precisa ser substituído pelo **ID real do sistema** Starfinder 2e no Foundry.
2. Revise os links do GitHub caso o nome de usuário ou o repositório mudem.
3. Substitua os arquivos de exemplo em `translation/` pelas chaves reais do sistema.
